## About Me

I am a **Junior AI and ML Engineer** focused on building practical, reliable AI systems.  
My experience includes:

- Developing **agent-based systems** using ADK and LangGraph  
- Building backend APIs and services with **FastAPI**  
- Deploying applications with **Docker and Kubernetes**  
- Applying deep learning models to real-world projects  
- Understanding and practicing **system design** for scalable AI applications  

I am currently expanding my knowledge of **monitoring and observability**, but I already work confidently with deployment, architecture, and end-to-end system implementation.

My philosophy is simple: build consistently, understand the whole system, and learn through real projects.

## Current Projects

### Agent Development with ADK  
Designed and deployed an AI agent using the Agent Developer Kit.  
This includes agent lifecycle management, message routing, tool integration, and modular workflow design.

### VectorMind AI  
A full Retrieval-Augmented Generation system using ChromaDB and FastAPI, containerized with Docker and designed for scalable environments.  
Strengthened my understanding of vector search, backend engineering, and deployment.

### Optimizers From Scratch  
Implemented key optimization algorithms such as Momentum, RMSProp, and Adam.  
Improved my understanding of gradient mechanics and deep learning frameworks.

### Applied ML Projects  
Projects include image classification (Brain Tumor), fake news detection, and other end-to-end ML systems.  
Each project includes data processing, modeling, evaluation, and interfaces for real usability.
